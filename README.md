# Inspecteur HTML

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live-2ea44f?logo=github)](https://grafikart.github.io/visual-inspector/)

JavaScript simple pour ajouter un overlay pour inspecter les polices et espacements d'une maquette (pour les travaux pratiques de la formation CSS)

```html
<script type="module" crossorigin src="https://grafikart.github.io/visual-inspector/inspector.js"></script>
```
